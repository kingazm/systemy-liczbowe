System ósemkowy (oktalny)

System pozycyjny o podstawie 8, do zapisu w nim liczb używa się ośmiu pierwszych cyfr arabskich:

0, 1, 2, 3, 4, 5, 6, 7

[Przykład liczby: 123078/12307O]

Konwersja

Zamiany z systemu decymalnego na oktalny dokonuje się poprzez dzielenie liczby przez 8, zapisywanie reszt, a następnie dzielenie przez 8 wyniku poprzedniego dzielenia.

Wpisz poniżej liczbę całkowitą w systemie decymalnym, aby zobaczyć jej postać w systemie ósemkowym

Natomiast konwersja z systemu oktalnego na system decymalny polega na dodawaniu jej cyfr.

Wpisz poniżej liczbę całkowitą w systemie ósemkowym, aby zobaczyć jej postać w systemie dziesiętnym

Zastosowanie

System ósemkowy jest stosowany w informatyce, choć nieco rzadziej niż system heksadecymalny. Przykładem jego zastosowania jest ustawianie praw do pliku danego obiektu w systemie Linux i językach programowania takich, jak C, C++, Java czy PHP.
